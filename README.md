# Smart Brain API
Back-end Node.js server for smart-brain facial recognition app. Handles client routing, user data management and Clarifai API requests. 

Technologies:

- Node.js Server
- Express.js
- Knex.js
- PostgreSQL
- Deployed on Heroku
